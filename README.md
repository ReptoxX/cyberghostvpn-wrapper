## Usage

```bash
cgw connect
```

## Installation

```bash
sudo curl -L https://github.com/ReptoxX/cyberghostvpn-wrapper/releases/download/v1.0.1/cgw -o /usr/local/bin/cgw \
&& sudo chmod +x /usr/local/bin/cgw
```
